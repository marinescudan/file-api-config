# The ultimate project

Node Express Pug project

## Development

Install project
Start mongodb and the start the node server
```
npm install
```
Start Development server
```
set app_password=****
node index 
// or
nodemon index
```
Open web interface
```
http://localhost:4000/view/home
http://localhost:4000/view/<model>
```
Access web api
```
http://localhost:4000/api/<model>
```
Enjoy!
